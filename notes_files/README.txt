# Notes Files Storage

This folder contains all the PDF notes files for Engineering Mathematics topics.

## How to Add Notes Files

1. Place your PDF files in this folder
2. Name them exactly as referenced in Notes.html:
   - Differential_Calculus.pdf
   - Integral_Calculus.pdf
   - Differential_Equations.pdf
   - Successive_Differentiation.pdf
   - Linear_Algebra.pdf
   - Numerical_Methods.pdf
   - Laplace_Transform.pdf
   - Fourier_Series.pdf
   - Partial_Differential_Equations.pdf
   - Vector_Calculus.pdf
   - Complex_Analysis.pdf
   - Fourier_Transform.pdf

## File Organization

Keep all PDF files in this 'notes_files' folder. The download links in Notes.html will automatically connect to files in this directory.

Example file path: `d:\Tutor\My_Lectures_Youtube\notes_files\Differential_Calculus.pdf`
